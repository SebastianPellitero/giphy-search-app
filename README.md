
## Giphy Search App


## Build Instructions
- Clone the repo
```
git clone git@github.com:SebastianPellitero/giphy-search-app.git

```
- Install dependencies
```
npm install
```
- Run the project
```
npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
